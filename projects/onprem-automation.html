On-Prem Server Maintenance Automation

Technologies: Ninja

Overview

Used NinjaOne tool to automate software patching along with OS patching. I was able to streamline this and now patching occurs once a month after its weekly reboot. The process scans the patch, installs the following week, and then applies the last week.

Key Features

  • Automated software and OS patching
  • Use NinjaOne tool and followed reboot schedule
  • Streamlined manual intervention and patching on basic updates like Google and Windows updates

Outcome

The project successfully streamlined the process of patching monthly software and OS patches that would occur each month. This has saved the company close to 20 or 30 hours a month of labor due to the amount of servers we have.
